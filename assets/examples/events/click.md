## Click event

When an HTML item has been clicked

### Javascript

```html
<script>
  document.addEventListener("click", function (event) {
    alert("Js vanilla code");
  });
</script>
```

#### Example:

<button class="e1Js">Click Me!</button>

### jQuery

```html
<script>
  $(document).ready(function () {
    alert("jQuery code");
  });
</script>
```

#### Example:

<button class="e1JQuery">Click Me!</button>
